❄️ Round Robin CPU Scheduler - Frosty Edition
An interactive pedagogical tool developed to visualize and simulate the Round Robin (RR) CPU scheduling algorithm.

🚀 Core Features
Dynamic Simulation: Input custom Arrival Times (AT), Burst Times (BT), and Time Quanta (QT).

Automated Analytics: Calculates Completion Time (CT), Turnaround Time (TAT), Waiting Time (WT), and Response Time (RT) automatically.

Visual Timelines: Generates a real-time Gantt Chart and Ready Queue history for every execution.

Interactive Game Mode: A "Network Access Game" that simulates 5-15 players competing for resources using RR logic.

Comparison Engine: Side-by-side technical comparison with FCFS, SJF, and Priority scheduling algorithms.

🛠️ Tech Stack
Frontend: HTML5 & CSS3 (Custom animations and "Frosty" winter theme).

Logic: Vanilla JavaScript (Pure algorithmic implementation).

Typography: Orbitron & Sans-serif fonts.

📊 Technical Implementation
The project implements the preemptive nature of Round Robin by:

Managing a circular Ready Queue.

Handling context switching based on the defined Time Quantum.

Calculating average performance metrics to evaluate system efficiency.
